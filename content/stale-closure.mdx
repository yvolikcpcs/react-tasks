# Fix the Counter

The counter in `setInterval` always sees the initial value of `count` because of a **stale closure**. 
Use the functional update pattern to fix it.

<InteractiveTask 
  taskTitle="Stale Closure" 
  solution="setCount(c => c + 1)"
>

```jsx
function Counter() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    const id = setInterval(() => {
      setCount(count + 1);
    }, 1000);
    return () => clearInterval(id);
  }, []);

  return <h1>{count}</h1>;
}
```
