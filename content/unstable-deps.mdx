# Stop the Infinite Effect

The effect re-runs on every render because `options` is a new object each time.
Stabilize dependencies to avoid repeated fetches.

<InteractiveTask
  taskTitle="Unstable Dependencies"
  solution="Memoize options with useMemo or move it outside the component."
>

```jsx
function Inventory({ client, warehouseId }) {
  const options = { includeBackorder: true };
  const [items, setItems] = useState([]);

  useEffect(() => {
    client.getInventory(warehouseId, options).then(setItems);
  }, [client, warehouseId, options]);

  return (
    <section>
      <h2>Inventory</h2>
      <ul>
        {items.map(item => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </section>
  );
}
```
</InteractiveTask>
