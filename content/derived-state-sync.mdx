# Fix the Derived State

The component stores a filtered list in state and updates it in an effect, but it
drifts out of sync when `items` changes quickly. Remove the redundant state and
derive the list from source data instead.

<InteractiveTask
  taskTitle="Derived State Drift"
  solution="const visible = useMemo(() => items.filter(...), [items, query])"
>

```jsx
function ItemList({ items, query }) {
  const [visible, setVisible] = useState([]);

  useEffect(() => {
    const next = items.filter(item =>
      item.label.toLowerCase().includes(query.trim().toLowerCase())
    );
    setVisible(next);
  }, [items, query]);

  const total = visible.length;

  return (
    <section>
      <h2>Items ({total})</h2>
      <ul>
        {visible.map(item => (
          <li key={item.id}>{item.label}</li>
        ))}
      </ul>
    </section>
  );
}
```
